AppCode快捷键

官方文档 《AppCode Quick Start Guide》

最近试着用AppCode写代码，记录下自己的快捷键

编辑

快捷键 使用说明
ctl + space 代码补全（类型、方法和变量）
ctl + shift + spcae 代码补全（联系前后文）
shift + cmd + enter 声明一行结束；
cmd + shit + i 查看函数的声明和实现
ctr + j 快速预览文档
ctl + enter,ctr + n 生成常用代码
cmd + mouse click 跳转到定义
cmd + shift + [/] 左右移动标签页
cmd + i 实现声明/协议的方法
alt + cmd + t 把选择代码放进代码片段里（if else autorelease{} … ）
shit + cmd + 上/下 上/下移代码
cmd + / 注释当前代码
shift + cmd + / 注释选中代码块
alt + enter 快速修复隐患和警告
alt + (shift+) cmd + l 检查(指定类型)隐患
cmd + o 重载父类方法
alt + f1 查看工程/文件/类型结构，在web/finder打开..
cmd + p 查看函数签名

重构

快捷键 使用说明
cmd + alt + c 重构-提取常量
cmd + alt + m 重构-选中代码为函数
cmd + alt + d 重构-选中常量为宏
cmd + alt + p 重构-选中变量增加函数参数
cmd + alt + v 重构-选中代码为变量
cmd + alt + f 重构-局部变量为实例变量
shift + F6 重构-重命名
cmd + alt + shift + t 重构菜单

搜索和替换

快捷键 使用说明
cmd + f 搜索
cmd + g 下一个
cmd + shift + g 上一个
cmd + r 替换
cmd + shift + f 指定路径搜索
cmd + shift + r 替换指定路径

编译和运行

快捷键 使用说明
cmd + F9 编译
ctr + r/d run/debug
ctr + alt + r/d 设置target并run/debug
cmd + . stop

导航跳转

快捷键 使用说明
cmd + shift + o 跳转指定符号
cmd + n 跳转到指定类型
cmd + shift + n 跳转到指定文件
cmd + ctr + 左/右 跳转前/后
cmd + e 最近打开的文件
cmd + ctr + 上 跳转头文件
ctr + 0~9 跳转书签
ctr + shit + 0~9 添加书签
cmd + F12 文件的代码结构
@property与@synthesize
