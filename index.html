编辑

快捷键 使用说明
ctl + space 代码补全（类型、方法和变量）
ctl + shift + spcae 代码补全（联系前后文）
shift + cmd + enter 声明一行结束；
cmd + shit + i 查看函数的声明和实现
ctr + j 快速预览文档
ctl + enter,ctr + n 生成常用代码

alt + enter 快速修复隐患和警告

cmd + o 重载父类方法
alt + f1 查看工程/文件/类型结构，在web/finder打开..
cmd + p 查看函数签名

重构

快捷键 使用说明
cmd + alt + c 重构-提取常量
cmd + alt + m 重构-选中代码为函数
cmd + alt + d 重构-选中常量为宏
cmd + alt + p 重构-选中变量增加函数参数

##2.setter语意

###什么时候使用这些语意特性呢？
只要是值类型、简单类型的类型，比如说NSInteger、CGPoint、CGFloat，以及C数据类型int、float、double等，都应该使用assign。
那么对于含有可深复制子类的对象，比如说NSArray、NSSet、NSDictionary、NSData、NSString等等，都应该使用copy特性。

注意：对于NSMutableArray之类的可变类型，不能够使用Copy特性，否则初始化会出现错误。

至于其他的NSObject对象，那么都应该使用retain来进行操作，这也是绝大多数所使用的情况。
